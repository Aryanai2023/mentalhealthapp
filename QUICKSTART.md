Get your Mental Health Companion app running in 3 simple steps!

## Step 1: Install Dependencies

Open your terminal in the project folder and run:

```bash
npm install
```

This will download all the necessary packages (React, Tailwind CSS, Vite, etc.)

## Step 2: Start the Development Server

```bash
npm run dev
```

## Step 3: Open Your Browser

The app will automatically open at `http://localhost:3000`

If it doesn't open automatically, just click the link in your terminal!
